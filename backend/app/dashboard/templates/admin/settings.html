Ollama Context Length

When enabled, MindRouter always overrides user-supplied num_ctx with the model's configured context length. This prevents users from oversubscribing backend GPU memory.

When enabled, MindRouter automatically generates descriptions for new models by searching the web for model cards and summarizing them with an LLM. Only models with no description are enriched.
