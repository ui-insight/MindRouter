API Usage

Users authenticate with their MindRouter API key. Each search deducts {{ quota_tokens }} tokens from their quota.

curl -X POST https://mindrouter.uidaho.edu/v1/search \
  -H "Authorization: Bearer $MINDROUTER_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"query": "latest advances in LLM inference", "max_results": 5}'

Also available at POST /api/search (same endpoint, Ollama-style path).
