API Usage

OpenAI-compatible endpoint:

curl -X POST https://mindrouter.uidaho.edu/v1/images/generations \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "model": "black-forest-labs/FLUX.2-dev",
    "prompt": "A mountain at sunset",
    "size": "1024x1024",
    "response_format": "b64_json"
  }'

Optional parameters:

  • n — number of images (max {{ max_n }})
  • size — dimensions (default: {{ default_size }})
  • num_inference_steps — quality steps (default: {{ default_steps }}, max: {{ max_steps }})
  • guidance_scale — prompt adherence (default: {{ default_guidance_scale }})
  • seed — reproducibility seed
  • response_format — url or b64_json
