API Usage

Send a multipart POST to /v1/ocr with your API key:

curl -X POST https://mindrouter.uidaho.edu/v1/ocr \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -F "file=@document.pdf" \
  -F "output_format=markdown"

Optional form fields: model, output_format (markdown|json), chunk_size, overlap, dpi
